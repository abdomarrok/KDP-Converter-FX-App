# Boilerplate FX App

## Overview
This is a boilerplate JavaFX application.

## Prerequisites
- Java 21 SDK
- Maven

## How to Run

### Using IDE (Recommended)
Open the project in IntelliJ IDEA or VS Code and run `com.boilerplate.app.App`.

### Using Command Line
If Maven is in your PATH:
```powershell
mvn clean javafx:run
```

If using the included batch file, edit `run.bat` to point to your Maven installation.
